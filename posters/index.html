使用步骤：

  1. 运行应用：npm run dev
  2. 浏览器打开应用，分别访问首页、添加记录、趋势、AI、日历页面
  3. 对每个页面截图，保存到 posters/screenshots/ 文件夹
  4. 截图文件名：home.png、add.png、stats.png、ai.png、calendar.png
  5. 刷新海报页面即可看到效果，然后浏览器截图保存海报
